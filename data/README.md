The structure of directory `./data` should be:
 
    data
    ├── mscoco
    │   ├── query.pkl
    │   ├── train.pkl
    │   └── retrieval.pkl
    ├── flickr25k
    │   ├── query.pkl
    │   ├── train.pkl
    │   └── retrieval.pkl
    └── nus-wide
        ├── query.pkl
        ├── train.pkl
        └── retrieval.pkl
